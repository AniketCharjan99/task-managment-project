# Enterprise Task Management System

A scalable **Task Management System** built using **Spring Boot Microservices and React** that helps teams manage projects, tasks, and collaboration efficiently.

The system allows users to **register, authenticate, create projects, assign tasks, track progress, and manage team workflows**. It is designed using **modern enterprise architecture** including JWT authentication, RESTful APIs, and service-to-service communication.

This project demonstrates how real-world enterprise applications are structured using **Java, Spring Boot, Microservices, MySQL, and React**.

The application supports **secure authentication, role-based access, project organization, task assignment, and notifications**.

This repository is intended for **learning enterprise backend architecture and building scalable SaaS-style applications similar to Jira, ClickUp, or Asana.**
